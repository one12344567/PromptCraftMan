提示词优化Agent

一款自动化、智能化的提示词优化智能体，可快速优化各类大模型提示词，提升提示词的精准度、逻辑性与指令执行效果，助力用户高效生成高质量Prompt。

核心功能

1. Prompt智能优化：针对模糊、冗余、逻辑混乱的原始提示词，自动梳理结构、精简表述、补充关键约束，生成更贴合大模型理解逻辑的优化版提示词。

2. 多场景适配：支持文案创作、代码编写、数据分析、问答咨询等多种使用场景的提示词优化，适配不同大模型交互需求。

3. 优化逻辑可追溯：清晰展示提示词优化思路与修改点，帮助用户理解优化逻辑，逐步提升自主编写Prompt的能力。

4. 轻量化运行：无需复杂配置，支持快速调用，可集成至各类AI工具、开发平台中。
